# 시간표짜조 — Frontend

> 과거 수강 이력과 선호 조건을 입력하면 AI가 최적의 시간표를 추천해주는 서비스

---

## 서비스 개요

학기마다 개설 과목을 학습한 AI가 가능한 모든 시간표 조합 중 사용자의 조건에 맞는 최적안을 선별합니다.  
사용자는 수강 신청 안내 책자 PDF와 과거 수강 이력(Excel)을 업로드하고, 원하는 조건을 설정하기만 하면 됩니다.

---

## 화면 구성

### 1. 로그인
이메일/비밀번호 로그인 및 카카오·구글 소셜 로그인 지원

### 2. 과거 수강 이력 업로드
- 과거 수강 내역 Excel 파일 업로드
- AI 학습 진행률 실시간 표시
- 학습 완료 항목 확인 (전공 과목 분류 / 개인 이수 학점 분류)

### 3. 시간표 선호도 입력
- **목표 취득 학점** — 슬라이더로 이번 학기 목표 학점 설정
- **현재 학년** — 1 · 2 · 3 · 4학년 중 선택
- **선호 공강 요일** — 수업 없이 비워두고 싶은 요일 다중 선택
- **1교시 수업 여부** — 오전 9시 수업 포함 여부 설정

### 4. AI 추천 시간표
- 조건에 맞는 시간표 조합을 주간 달력 형태로 시각화
- 정해진 점수에 따라 높은 점수의 시간표부터 2~3개 시간표 시각화
- 최종 시간표 확정

### 5. 학업 분석 리포트 *(예정)*
- 이수 구분별 현황 (전공 필수 / 전공 선택 / 공인어학)
- 전체 이수율 도넛 차트
- 학점 수강 과목 현황
