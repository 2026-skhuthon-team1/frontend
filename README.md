# 시간표짜조 — Frontend

> 과거 수강 이력과 선호 조건을 입력하면 AI가 최적의 시간표를 추천해주는 서비스

---

## 서비스 개요

학기마다 개설 과목을 학습한 AI가 가능한 모든 시간표 조합 중 사용자의 조건에 맞는 최적안을 선별합니다.  
사용자는 과거 수강 이력(Excel)을 업로드하고, 원하는 조건을 설정하기만 하면 됩니다.

---

## 화면 구성

### 1. 수강 이력 업로드 `/upload` (시작 화면)
- 학사정보시스템에서 받은 Excel 파일 드래그&드롭 업로드
- 분석 진행률 실시간 표시
- 4단계 분석 체크리스트
  - 엑셀 파일 읽기
  - 전공 · 교양 이수 학점 분류
  - 졸업 요건 대비 잔여 학점 계산
  - 들을 수 있는 과목 추리기

### 2. 시간표 선호도 입력 `/input`
- **전공 학점** — 슬라이더로 이번 학기 전공 학점 설정 (0~18학점)
- **교양 학점** — 슬라이더로 이번 학기 교양 학점 설정 (0~12학점)
- **현재 학년** — 1 · 2 · 3 · 4학년 중 선택
- **선호 공강 요일** — 월~금 중 다중 선택
- **1교시 수업 여부** — 오전 9시 수업 포함 여부 (가급적 피하기 / 상관 없음)
- **사회봉사 포함 여부** — 사회봉사 과목 시간표 포함 여부
- **전공 선택** — 드롭다운 선택, 복수전공 시 여러 개 선택 가능

### 3. AI 추천 시간표 `/result`
- AI 추천 조합 목록 (추천 순위 배지)
- 선택한 조합의 주간 달력 시간표 미리보기
- 수업 블록에 과목명 · 강의실 · 이수 구분(전선 / 전필 / 교양 / 사회봉사) 표시
- 시간표 이미지 저장 (PNG, 2x 해상도)
- 최종 선택 및 확정

### 4. 학업 분석 리포트 `/report` *(미구현)*
- 이수 구분별 현황 (전공 필수 / 전공 선택 / 교양)
- 전체 이수율 도넛 차트
- 학기별 수강 과목 현황

## 기술 스택

| 항목 | 사용 기술 |
|------|----------|
| UI | React 19 |
| 스타일 | Tailwind CSS v4 |
| 라우팅 | React Router v7 |
| 전역 상태 | Zustand v5 |
| 서버 상태 / API | TanStack React Query v5 + Axios |
| 빌드 / 개발 서버 | Vite 8 |
| 이미지 캡처 | html-to-image |
